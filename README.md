# LiteDownloader
its a simple download manager for android
## Feature
* its has simple ui
* you can pause and resume downloads
* you can concurrently download multiple file
* after completion you can open downloads
## How to use
clone this repo or download the zip file and open it in android studio and from there you can generate apk.
